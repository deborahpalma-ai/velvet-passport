# Velvet Passport ✈️

**Luxury Travel Planning & Global Currency Intelligence**

Velvet Passport is a premium SaaS application designed for travelers who want to organize every aspect of their trips while maintaining complete financial control across multiple currencies.

Unlike traditional travel planners, Velvet Passport combines itinerary management, budgeting, real-time currency conversion, weather forecasts, and temporary local events into a single elegant platform.

---

## ✨ Features

### 🔐 Authentication

- Secure user registration
- Login with email and password
- Password recovery
- Individual user accounts
- Row-Level Security (RLS)

---

### 🌍 Trip Planning

- Create unlimited trips
- Country, state/province and city selection
- Automatic trip duration calculation
- Daily itinerary planning
- Custom travel checklist

---

### 🏨 Accommodation

- Hotel registration
- Daily rate calculation
- Automatic total accommodation cost

---

### 🍽️ Food Planning

- Food budget
- Restaurants
- Bars
- Beach Clubs

---

### 🎯 Activities

- Attractions

- Personal activities
- Estimated costs

---

### 🚗 Transportation

- Flights
- Trains
- Bus

- Rental cars

### 💰 Financial Dashboard

Automatic calculation of:

- Accommodation
- Food
- Restaurants
- Bars
- Beach Clubs
- Activities
- Transportation

Displays:

- Total travel cost
- Spending by category
- Interactive charts
- Budget overview

---

### 💱 Global Currency Intelligence

Supports travelers worldwide.

Features include:

- Automatic destination currency detection
- User home currency selection
- Live exchange rates
- Multi-currency expense tracking
- Real-time currency conversion
- Historical exchange rate comparison
- Currency fluctuation monitoring

Example:

A user living in Ireland earning in EUR can instantly see how much a trip to Canada will cost in both EUR and CAD.


### 🌤 Smart Travel Assistant

Integrated AI provides:

- Weather forecast during travel dates
- Temperature expectations
- Packing recommendations
- Temporary local events
- Festivals
- Cultural events
- Sports events
- Concerts

The AI **does not** generate itineraries or recommend restaurants or hotels. It only provides contextual information related to the selected destination and travel dates.
